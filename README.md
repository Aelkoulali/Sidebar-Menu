# 🗂️ Responsive Sidebar Menu 🟰 
This Responsive Sidebar Menu is a web project built with HTML, CSS, and JavaScript. It uses an onclick event to open and close the sidebar, and it dynamically manages menu items using forEach(), along with add() and remove() methods to update the menu in real time.


## ✅ Key Features
- Responsive sidebar menu that works well on both desktop and mobile screens.
- Toggle open/close behavior for the sidebar using onclick events.
- Smooth interaction: menu items can be added and removed dynamically.
- Uses forEach() to loop through menu items and attach/update logic.
- Manages menu lists using add() and remove() methods (for adding/removing items from the menu/DOM).

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript: interactivity using onclick, forEach(), add(), and remove() to control menu behavior

## ▶️ How to Run
Open `index.html` in your browser.
